# 原神风物之诗琴模拟器
***Windsong-Lyre-Simulator by Mimangz*** *迷茫子的梦想启航项目！边做边学！*

## 使用说明：
加载完成后点击**小派蒙**进入演奏界面。

如果网络不稳定导致加载时间过长，我允许你**强行进入**，但是不推荐这样做！

可使用鼠标或者键盘演奏。键盘键位与原神键位一致，另支持数字键和小键盘。
